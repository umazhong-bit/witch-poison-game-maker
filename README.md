## ✨ 一句话玩法

> **读对才能吃，吃错可能中毒；先喝下三瓶毒药的小组失败。**

全班分成 A、B 两组，各自秘密藏好 3 张毒药。学生轮流选择词卡并大声认读，老师负责判断“读对”或“读错”。读对后卡片才会被吃掉；如果刚好是毒药，小组就会失去一格生命。

## 🧪 课堂里会发生什么？

```text
老师输入词表
      ↓
生成 5×5 魔法词卡棋盘
      ↓
A、B 两组秘密放置毒药
      ↓
学生选词并大声认读
      ↓
读对：吃掉卡片  │  读错：换一张再挑战
      ↓
看看谁能躲过女巫的毒药！
```

## 🎮 游戏特色

| 魔法道具 | 课堂作用 |
| --- | --- |
| 🃏 5×5 词卡棋盘 | 25 张卡片，词数不足时自动均匀重复 |
| ☠️ 双组秘密毒药 | A、B 两组各藏 3 张，可以选中相同位置且不会泄露对方选择 |
| 🧪 三格药水生命 | 吃到毒药失去一格，先清空生命的一组失败 |
| 👩‍🏫 教师裁判模式 | 不自动判音，老师掌握“读对/读错”和课堂节奏 |
| 🖥️ 教室大屏优化 | 支持横屏、全屏，棋盘和控制区无需滚动 |
| 📦 完全离线 | 图片、样式和脚本全部嵌入一个 HTML 文件 |

## 🚀 快速施法

无需安装第三方 Python 库：

```bash
python3 scripts/generate_game.py \
  --words "museum, school, train station, shop, cinema, park, mountain, library" \
  --output "/目标目录/女巫的毒药.html"
```

生成完成后，直接双击 HTML 文件即可开始游戏。脚本还会生成一个同名的 `.manifest.json` 文件，用来记录本次词表和规则版本。

## 📜 不可打破的课堂契约

- 学生必须先选卡并大声认读，读对后再次点击才算吃掉。
- 读错不会吃掉卡片，本组需要换另一张继续挑战。
- A、B 两组的毒药可以重合；若毒药全部吃完仍无人耗尽生命，则剩余生命多者获胜，同生命时平局。
- 不加入倒计时、自动语音识别或额外计分，让老师掌握节奏。
- 默认不依赖网络、CDN、外部字体或外部图片。

完整规则与 Skill 使用说明请查看 [`SKILL.md`](SKILL.md)。
